# Geyser Packet Logger Extension
This extension for GeyserMC allows you to log all incoming and outgoing packets between the Minecraft Bedrock client and the Minecraft Java server. It is useful for debugging, development, and understanding the communication between the two versions of Minecraft.

## Features
- Log packets to files for later analysis
- Web interface to view logged packets in real-time
